# Venice Island Cables 🌊

Програма обчислює **мінімальну довжину підводних кабелів**, необхідних
для з'єднання всіх островів Венеції в єдину мережу.

## Алгоритм

Задача зводиться до пошуку **мінімального кістякового дерева (MST)**
у зваженому незорієнтованому графі. Використовується **алгоритм Прима**
з пріоритетною чергою (heap) — складність `O(E log V)`.

## Структура проєкту

```
venice_cables/
├── main.py       # Точка входу
├── graph.py      # Зчитування матриці суміжності з CSV
├── mst.py        # Алгоритм Прима (MST)
├── tests.py      # Юніт-тести (pytest)
├── islands.csv   # Приклад вхідних даних (9 островів)
└── README.md
```

## Вхідні дані

Файл `islands.csv` — матриця суміжності розміром N×N, де:
- `matrix[i][j]` — відстань між островами `i` та `j`
- `0` означає відсутність прямого з'єднання
- `1 ≤ N ≤ 100`

**Приклад `islands.csv` для 3 островів:**
```
0,4,8
4,0,3
8,3,0
```

## Запуск

```bash
python main.py islands.csv
```

**Вивід:**
```
Мінімальна довжина підводних кабелів: 37.0
```
